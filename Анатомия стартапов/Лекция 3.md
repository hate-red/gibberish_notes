### Потребительские сегменты
Остервальдер выделяет 5 потребительских сегментов:
1. **Массовый** рынок - универсальное предложение
2. **Нишевый** рынок - глубокая специализация
3. **Сегментированный** рынок - несколько вариантов предложения
4. **Диверсифицированный** рынок - несколько независимых моделей
5. **Многосторонняя** платформа - ценность в соединении групп

Группы клиентов представляют различные сегменты, если:
- Различия в их **запросах** обуславливают различия в **предложениях**
- Взаимодействие осуществляется по разным **каналам сбыта**
- **Взаимоотношения** с ними нужно строить по-разному
- Их **выгодность** существенно различается
- Их привлекают **разные аспекты** предложения

### Ценностное предложение
*Ценностное предложение* - это причина, по которой клиенты отдают предпочтение одной компании перед другой

*Ценностное предложение* - это совокупность преимуществ, которые компания готова предложить потребителю

Элементы ценностного предложения:
1. Новизна
2. Кастомизация
3. Результат
4. Производительность
5. Дизайн
6. Удобство
7. Доступность
8. Экономия
9. Бренд
10. Безопасность
11. Цена

### Каналы сбыта
Каналы **связи**, **распространения** и **продаж** составляют систему взаимодействия компании с клиентом.

Ключевые функции каналов:
1. **Информирование** (повышение осведомленности о продукте)
2. **Оценка** (помощь клиенту в оценке ценностного предложения)
3. **Покупка**
4. **Доставка**
5. **Послепродажное обслуживание**

Каналы сбыта по типу владения:
1. **Прямые** (собственный сайт, отдел продаж)
2. **Косвенные** (маркетплейсы, дилеры)
3. **Гибридные**

Каналы сбыта по характеру взаимодействия:
1. **Онлайн**
2. **Оффлайн**

### Взаимоотношения с клиентами
Мотивы:
1. Привлечение
2. Удержание
3. Стимулирование повторных покупок

| Тип отношений                | Уровень вовлечения | Затратность   | Пример                        |
| ---------------------------- | ------------------ | ------------- | ----------------------------- |
| Сомообслуживание             | Низкий             | Низкая        | Супермаркет                   |
| Автоматизированное           | Средний            | Низкая        | Netflix                       |
| Сообщества                   | Высокий            | Средняя       | Github                        |
| Персональная поддержка       | Высокий            | Высокая       | Менеджер по ключевым клиентам |
| Специализированная поддержка | Очень высокий      | Очень высокая | Адвокат                       |
| Со-создание                  | Максимальный       | Переменная    | Youtube                       |

### Потоки доходов

Типы доходов:
1. Доход от **разовых сделок**
2. Регулярный доход от **периодический платежей**

Способы генерации дохода:
1. Разовая продажа
2. Плата за использование
3. Плата за подписку
4. Аренда/лизинг
5. Лицензирование
6. Комиссия
7. Реклама

### Ключевые ресурсы
*Ключевые ресурсы* - это наиболее важные активы, необходимые для функционирования и масштабирования бизнес-модели.

Категории ключевых ресурсов:
1. Материальные
2. Интеллектуальные
3. Человеческие
4. Финансовые

Виды ключевых ресурсов:
1. Уникальные патенты
2. Сильный бренд
3. Экспертные знания
4. Талантливая команда
5. Сеть дистрибуции
6. Производственное оборудование
7. Денежные средства
8. Сырье и материалы

### Ключевые виды деятельности
*Ключевые виды деятельности* - это стратегически важные операции и действия, которые компания должна выполнять, чтобы ее бизнес-модель была успешной.
К ним относят:
1. Производства
2. Разрешение проблем
3. Платформа/сети

### Ключевые партнеры
*Партнерства* - это стратегические союзы, которые позволяют компании:
1. Снижать риски и неопределенность
2. Получать доступ к ключевым ресурсам и технологиям
3. Достигать эффекта масштаба и оптимизировать издержки
4. Создавать Комплексные ценностные предложения

Виды партнерств:
1. Стратегическое сотрудничество
2. Со-конкуренция
3. Совместные предприятия
4. Отношения производителя с поставщиками

### Структура издержек
*Издержки* - расходы, связанные с функционированием бизнес-модели

По типам издержек:
1. **Cost**-driven (минимизация затрат)
2. **Value**-driven (максимизация качества)
