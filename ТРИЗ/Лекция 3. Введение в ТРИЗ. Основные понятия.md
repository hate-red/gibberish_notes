==*ТРИЗ*== - методология, объединяющая организованный, систематический метод решения проблем с анализом и прогнозированием, основанными на изучении закономерности изобретений в мировой патентной литературе 

###### Ключевые принципы ТРИЗ
1. Проблемы и решения повторяются в разных областях
2. Закономерности технической эволюции воспроизводятся (можно предсказывать будущие инновации)
3. Инновации имеют научные эффекты за пределами своей области (можно применить инновацию и к другой области)
4. Систематический подход к инновациям (можно задать план развития и улучшений)

###### Зачем ТРИЗ в системной инженерии?
- Систематизация процессов
- Решение сложных проблем
- Улучшение качества кода
- Инновации в разработке

### Основные понятия
==*Техническая система*== - совокупность взаимосвязанных компонентов (подсистем, предназначенных для выполнения определенной функции

 Ключевые аспекты:
1. Структура и взаимодействие элементов
2. Выявление точек для улучшения (сперва устраняется самая большая просадка)
3. Обнаружение противоречий
4. Прогнозирование развития

==*Функция*== - действие, которые выполняет система или ее элемент

Классификация:
1. Полезные
2. Врезные
3. Нейтральные

**Функциональный анализ** позволяет понять как работает система, найти вредные функции

==*Противоречие*== - ситуация, когда улучшение одной характеристики приводит к ухудшению другой

Типы противоречий:
1. **Административное** (что-то работает плохо, но не понятно что именно)
2. **Техническое** (между двумя параметрами системы)
3. **Физическое** (один элемент должен обладать противоречивыми свойствами)

==*ИКР (идеальный конечный результат)*== - идеальное решение, при котором желаемый результат достигается сам по себе

Характеристики:
1. Устраняет недостатки исходной системы
2. Сохраняет преимущества исходной системы
3. Не усложняет систему и не создает новых недостатков
4. Система выполняет функцию без участия самого иззобретателя

==*Ресурсы*== - все, что может быть использовано для решения задачи и улучшения системы

Основные типы:
1. **Вещественные** (материалы, объекты)
2. **Полевые** (энергия)
3. **Информационные** (данные, знания, опыт, алгоритмы)
4. **Временные** (использование времени, его использования)
5. **Пространственный** (форма, размер, расположение)
6. **Функциональный** (способности элементов системы выполнять полезные действия)
