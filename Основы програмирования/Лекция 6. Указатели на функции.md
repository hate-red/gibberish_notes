*Указатель на функцию* - это переменная, которая хранит адрес функции в памяти
##### Указатель на функцию позволяет
1. Вызвать функцию не по имени, а через указатель
2. Хранить функции в структурах данных
3. Передавать функции как аргументы другим функциям (callback)
4. Возвращать функции из функций

>Синтаксис:

```C++
// тип_результата (*имя_указателя)(типы параметров);
int (*f_ptr)(int, double);
f_ptr = func;

// Можно сразу так:
int (*f_ptr)(int, double) = func;

// Вызов
f_ptr(1, 2.3);
```


### Массивы указателей на функции

```C++
int f1(double);
int f2(double);
int f3(double);

int main() {
	int (*arr[])() = {f1, f2, f3};
}
```

### Функции как аргументы функций

```C++
void func(double (*f_ptr)(double), ...) {
	auto a = f_ptr(...);
}

double f1(...) {...}
double f2(...) {...}

int main() {
	func(f1);
	func(f2);
} 
```

>Функции можно передавать в другие функции для проверки условия (такая функция называется предикатом):

```C++
void func(bool (*f_ptr)(double), ...) {
	if (f_ptr(...)) {
		...
	}
}
```

>Функции можно передавать в другие функции для сравнения элементов массива, строк и т.д. (такая функция называется компаратором):

```C++
void func(int a, int b, bool (*f_ptr)(int, int)) {
	if (f_ptr(a, b)) {
		...
	}
}
```

### Typedef / using

>Typedef используется для объявления типа, чтобы не писать много повторяющегося кода:

```C++
typedef int (*FuncPtr)(int);

int main() {
	FuncPtr f_ptr1 = func1;
	FuncPtr f_ptr2 = func2;
}
```

>А еще есть using, с ним можно сделать тоже самое:

```C++
// Тут тип стоит слева от равенства, что нагляднее
using FuncPtr = int(*)(int);

int main() {
	// тоже самое
	FuncPtr f_ptr = func;
}
```

### Современные альтернативы

##### std::function

>std::function позволяет создать указатель на функцию, только ~~без богомерзких скобочек и звездочек~~ безопаснее и со всякими методами:

```C++
// function<что_вернет(что_передаем)>
function<int(double)> f_ptr = func;

f_ptr(1.23);
```

##### Лямбда-выражения (лямбда-функции)

>С помощью лямбд удобно писать анонимные функции, для быстрого использования

```C++
int main() {
	// Объявляем прямо в main'е (!)
	
	// -> что_вернет можно опустить, если очевидно, 
	// какой тип вернет функция
	auto lambdaSquare = [](int x) -> int {return x * x};
	cout << lambdaSquare(5); // 25
}
```
