# Что это?
Это небольшой репозиторий, где собраны мои конспекты лекций. Одна папка - один предмет. 
Внутри файлы в формате markdown (.md) и иногда дополнительные папи и файлы (например, папка Materials в ТРИЗ'е).

# Что с этим не так?
Так сложилось, что почти все свои лекции я пишу в Obsidian'е. 
Мне это удобно, но некоторые особенности форматирования markdown'а в obsidian'е не поддерживаются на github'е.
Вот некоторые из них:

### Выделеление текста цветом
В obsidian'е я пишу "==Термин==" и то, что находится между знаков равно выделяется цветом. Вот так:

Здесь такого нет. Оно никак не форматируется.
