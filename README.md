# Что это?
Это небольшой репозиторий, где собраны мои конспекты лекций. Одна папка - один предмет. 
Внутри файлы в формате markdown (.md) и иногда дополнительные папи и файлы (например, папка Materials в ТРИЗ'е).

# Что с этим не так?
Некоторые особенности форматирования markdown'а в obsidian'е не поддерживаются на github'е.

# Что с этим делать?
Вижу 2 варианта:
1. Когда видите что-то вроде ![[картинка.png]], поискать файлик с таким названием (картинка.png) в папке предмета.
2. Можете скачать себе Obsidian, сделать `git clone`, открыть скачанное, начать пользоваться.
